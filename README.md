# Rezka Filter

Расширение для браузера (Chromium), добавляющее удобные фильтры для списков сериалов на сайте `rezka.fi`. Позволяет скрывать сериалы определенных стран производства и поддерживает "бесконечную ленту", автоматически подгружая новые карточки сериалов взамен скрытых.

## Особенности

- **Фильтрация по странам:** Скрывайте сериалы из нежелательных стран одним кликом.
- **Бесконечная лента (авто-пагинация):** Расширение само подгружает следующие страницы сайта в фоновом режиме, пока на экране не наберется желаемое количество карточек сериалов. Вы больше не увидите пустых страниц после применения жестких фильтров!
- **Сохранение настроек:** Выбранные страны и желаемое количество карточек сохраняются в браузере и автоматически применяются при каждом посещении сайта.
- **Интеграция в дизайн:** Фильтр оформлен в виде компактной боковой панели с темной темой, не ломает сетку сериалов при подгрузке новых рядов.

## Установка (для разработчиков и ручной установки)

Так как расширение не опубликовано в Chrome Web Store, вы можете установить его вручную в режиме разработчика:

1. Скачайте или склонируйте этот репозиторий к себе на компьютер.
2. Откройте браузер на базе Chromium (Google Chrome, Яндекс Браузер, Microsoft Edge, Brave, Opera).
3. Перейдите на страницу управления расширениями:
   - В Chrome: `chrome://extensions/`
   - В Edge: `edge://extensions/`
   - В Яндекс Браузере: `browser://extensions/`
4. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
5. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
6. Выберите папку с исходным кодом расширения (папка, где находится файл `manifest.json`).

Готово! Теперь зайдите на страницу с топами сериалов на `rezka.fi` (например, `https://rezka.fi/series/best/`), и справа появится панель фильтров.

## Структура проекта

- `manifest.json` — конфигурационный файл расширения (Manifest V3).
- `main.js` — основной скрипт: логика авто-подгрузки (fetch), MutationObserver, инициализация.
- `parser.js` — модуль извлечения информации (стран) из HTML-карточек сериалов.
- `ui.js` — модуль отрисовки панели управления с использованием Shadow DOM для изоляции стилей.
- `storage.js` — модуль работы с `chrome.storage.local` для сохранения пользовательских настроек.
- `styles.css` — стили расширения (в том числе класс для скрытия карточек).

## Разработка

Проект использует Vanilla JavaScript и CSS без сборщиков, что делает его простым для изучения и модификации. Иконки создаются динамически (или используются заглушки) в папке `icons/`.
